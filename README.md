## 서비스 소개
### 서비스 개요
경북대학교와 그 아래 수많은 웹 페이지에 올라오는 수많은 정보를 자동으로 수집하고, 그 중 내게 꼭 필요한 정보를 강조하여 알려주는 웹 사이트

### 타서비스와의 차별점
기존처럼 여러 단과대학 및 단체별 하위 사이트를 헤메며 공지사항을 찾지 않아도 된다.
